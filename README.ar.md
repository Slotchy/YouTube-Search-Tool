# 📺 YouTube Search Tool

أداة ويب للبحث عن مقاطع فيديو وبث مباشر على YouTube بترتيب زمني حقيقي باستخدام مفتاح API الخاص بك. لا يلزم تسجيل الدخول — فقط الصق مفتاحك وابحث.

## **YouTube Search Tool:** <https://slotchy.github.io/YouTube-Search-Tool>
## **درس تعليمي لمفتاح API:** <https://www.youtube.com/watch?v=eE5WGiwqlFE>

---

## ✨ الميزات

* 🔍 ابحث عن مقاطع الفيديو أو البث المباشر مرتبة من الأحدث إلى الأقدم
* 📅 تصفية حسب التاريخ — اختر يومًا واحدًا أو نطاقًا مخصصًا للتواريخ باستخدام تقويم تفاعلي
* 🕐 تصفية حسب الساعة — تضييق النتائج إلى نافذة زمنية محددة ضمن التاريخ المحدد (يحول منطقتك الزمنية المحلية تلقائيًا إلى UTC)
* 🚫 تصفية Shorts — يخفي مقاطع الفيديو التي تحتوي على علامات التصنيف (`#`) في عنوانها أو وصفها
* ⏱️ تصفية مدة الفيديو — عند تحديد Videos، يمكن التصفية حسب قصير (<4 دقائق)، متوسط (4–20 دقيقة)، أو طويل (20+ دقيقة)
* 🎨 8 سمات لونية — بنفسجي، سماوي، زمردي، وردي، عنبري، أزرق، زهري، وليموني عبر لوحة عينات
* 💾 يتم حفظ مفتاح API وسمة الألوان في متصفحك تلقائيًا
* ✨ خلفية جسيمات متحركة، محملة مسبقًا عند الفتح
* 🕒 يعرض الوقت المنقضي منذ نشر كل نتيجة
* 🖱️ انقر على أي نتيجة لفتح الفيديو مباشرة

> **ملاحظة:** وضع البث المباشر معطل عند تفعيل تصفية التاريخ. تصفية مدة الفيديو متاحة فقط في وضع Videos.

## 🔑 كيفية الحصول على مفتاح YouTube API

1. افتح علامة تبويب جديدة، انتقل إلى [Gmail](https://mail.google.com) وسجّل الدخول إلى حساب Google الخاص بك
2. انتقل إلى [console.cloud.google.com](https://console.cloud.google.com)
3. عند المطالبة، اقبل **شروط الخدمة** وانقر على **أوافق وأتابع**
4. في أعلى يسار الصفحة، انقر على **تحديد مشروع**، ثم في أعلى يمين تلك النافذة انقر على **مشروع جديد**
5. امنح مشروعك اسمًا وانقر على **إنشاء** — انتظر حتى يكتمل الإنشاء
6. افتح **قائمة التنقل** (الأشرطة الثلاثة في أعلى اليسار)، مرر لأسفل إلى **واجهات برمجة التطبيقات والخدمات**، وانقر على **المكتبة**
7. مرر لأسفل حتى تجد **YouTube Data API v3** وانقر عليها
8. انقر على **تفعيل** وانتظر حتى يكتمل التفعيل
9. على الجانب الأيسر، انقر على **بيانات الاعتماد**
10. في الأعلى، انقر على **إنشاء بيانات اعتماد ← مفتاح API**، ثم انقر على **إنشاء**
11. بعد إنشاء المفتاح، انقر على **مفتاح API** لفتح إعداداته
12. تحت **قيود API**، انقر على **تقييد المفتاح**، ثم انقر على **تحديد واجهات برمجة التطبيقات**
    - إذا لم تظهر YouTube Data API v3 فورًا، أعد تحميل الصفحة
    - مرر لأسفل، حدد **YouTube Data API v3**، وانقر على **موافق**
13. انقر على **حفظ**
14. انقر على **إظهار المفتاح**، ثم انسخ مفتاحك
15. انتقل إلى [YouTube Search Tool](https://slotchy.github.io/YouTube-Search-Tool)، الصق مفتاحك وقم بإجراء بحث للتأكد من أنه يعمل

> **الحصة المجانية:** يوفر المستوى المجاني 10,000 وحدة/يوم. كل بحث يكلف ~100 وحدة (~100 بحث/يوم مجانًا).

## 🚀 طريقة الاستخدام

1. الصق مفتاح YouTube API الخاص بك — يتم حفظه تلقائيًا للزيارات المستقبلية
2. اختر **Videos** أو **Livestreams**
3. اكتب موضوع بحثك
4. *(اختياري)* حدد **مدة الفيديو** للتصفية حسب المدة (وضع Videos فقط)
5. *(اختياري)* ضع علامة على **تصفية Shorts** لاستبعاد مقاطع الفيديو التي تحتوي على علامات تصنيف في عنوانها أو وصفها
6. *(اختياري)* فعّل **تصفية حسب التاريخ** وحدد يومًا أو نطاقًا في التقويم
7. *(اختياري)* إذا كان التاريخ محددًا، عيّن ساعة **البدء** و**الانتهاء** للتصفية حسب وقت اليوم
8. انقر على **البحث عن الأحدث** (أو **بحث** عند تفعيل تصفية التاريخ)
9. انقر على أي بطاقة نتيجة لفتح الفيديو
10. استخدم عينات الألوان على اليمين لتغيير سمة التمييز — يتم تذكرها بين الزيارات

## 🔒 الخصوصية

* يتم تخزين مفتاح API محليًا في متصفحك (`localStorage`) — لا يُرسَل إلى أي مكان سوى Google
* يتم تخزين تفضيل سمة الألوان محليًا في متصفحك
* لا يوجد خادم خلفي — جميع المعالجات تتم على جانب العميل
* الطلبات الخارجية الوحيدة هي إلى Google YouTube Data API v3

## 🛠️ التفاصيل التقنية

| التفاصيل | المعلومات |
| --- | --- |
| المكدس | HTML، CSS، JavaScript نقي — بدون أطر عمل |
| الخادم الخلفي | لا يوجد — كامل على جانب العميل |
| واجهة برمجة التطبيقات | YouTube Data API v3 |
| النتائج | حتى 50، مرتبة حسب التاريخ |
| تصفية التاريخ | يستخدم معاملات `publishedAfter` / `publishedBefore` |
| تصفية المدة | يستخدم معامل `videoDuration` (`short` / `medium` / `long`) |
| تصفية Shorts | على جانب العميل — يزيل أي نتيجة تحتوي على `#` في العنوان أو الوصف |
| السمات | خصائص CSS المخصصة (`var(--accent)`) يتم تبديلها عبر JavaScript |
| الجسيمات | قيم `animation-delay` سالبة لتعبئة الجسيمات مسبقًا عند التحميل |
| التخزين | `localStorage` لمفتاح API وتفضيل السمة |
