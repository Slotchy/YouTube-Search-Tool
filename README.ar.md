# 📺 أداة البحث في يوتيوب

أداة قائمة على الويب للبحث في مقاطع الفيديو والبث المباشر على يوتيوب بترتيب زمني حقيقي باستخدام مفتاح YouTube API الخاص بك. لا يلزم تسجيل الدخول — فقط الصق مفتاحك وابحث.

## **أداة البحث في يوتيوب:** https://slotchy.github.io/YouTube-Search-Tool

---

## ✨ المميزات

* 🔍 البحث في مقاطع الفيديو أو البث المباشر مرتبةً من الأحدث إلى الأقدم
* 📅 التصفية حسب التاريخ — اختر يومًا واحدًا أو نطاقًا زمنيًا مخصصًا باستخدام تقويم تفاعلي
* 🕐 فلتر الساعة — تضييق النتائج إلى نافذة زمنية محددة ضمن التاريخ المحدد (يحوّل منطقتك الزمنية المحلية إلى UTC تلقائيًا)
* 🚫 تصفية Shorts — إخفاء مقاطع الفيديو التي تحتوي على وسوم (`#`) في عنوانها أو وصفها
* ⏱️ فلتر مدة الفيديو — عند تحديد وضع الفيديو، صفِّ حسب قصير (<4 دقائق)، متوسط (4–20 دقيقة)، أو طويل (20+ دقيقة)
* 🎨 8 ألوان للسمات — البنفسجي، الفيروزي، الزمردي، الوردي، الكهرماني، الأزرق، الوردي الغامق، والليموني عبر لوحة ألوان
* 💾 يتم حفظ مفتاح API وسمة الألوان في متصفحك تلقائيًا
* ✨ خلفية جسيمات متحركة، تُملأ مسبقًا عند التحميل
* 🕒 يعرض الوقت المنقضي لكل نتيجة
* 🖱️ انقر على أي نتيجة لفتح الفيديو مباشرةً

> **ملاحظة:** يتم تعطيل وضع البث المباشر عند تفعيل فلتر التاريخ. فلتر مدة الفيديو متاح في وضع الفيديو فقط.

---

## 🔑 كيفية الحصول على مفتاح YouTube API

1. اذهب إلى [console.cloud.google.com](https://console.cloud.google.com)
2. أنشئ مشروعًا جديدًا أو اختر مشروعًا موجودًا
3. فعِّل **YouTube Data API v3**
4. انتقل إلى **Credentials → Create Credentials → API Key**
5. انسخ مفتاح API الخاص بك والصقه في الأداة

> **الحصة:** الطبقة المجانية توفر 10,000 وحدة يوميًا. كل بحث يكلّف ~100 وحدة (~100 عملية بحث يوميًا مجانًا).

## 🚀 طريقة الاستخدام

1. الصق مفتاح YouTube API الخاص بك — يتم حفظه تلقائيًا للزيارات المستقبلية
2. اختر **مقاطع الفيديو** أو **البث المباشر**
3. اكتب موضوع البحث
4. *(اختياري)* اختر **مدة الفيديو** للتصفية حسب المدة (وضع الفيديو فقط)
5. *(اختياري)* ضع علامة على **تصفية Shorts** لاستبعاد الفيديوهات التي تحتوي على وسوم في عنوانها أو وصفها
6. *(اختياري)* فعِّل **التصفية حسب التاريخ** واختر يومًا أو نطاقًا من التقويم
7. *(اختياري)* إذا تم اختيار تاريخ، حدد ساعة **البداية** و**النهاية** للتصفية حسب وقت اليوم
8. انقر على **البحث عن الأحدث** (أو **بحث** عند تفعيل فلتر التاريخ)
9. انقر على أي بطاقة نتيجة لفتح الفيديو
10. استخدم لوحة الألوان على اليمين لتغيير السمة — يتم تذكرها بين الزيارات

---

## 🔒 الخصوصية

* يتم تخزين مفتاح API محليًا في متصفحك (`localStorage`) — لا يُرسل إلى أي مكان إلا إلى Google
* يتم تخزين تفضيل سمة الألوان محليًا في متصفحك
* لا يوجد خادم خلفي — تتم جميع المعالجة على جانب العميل
* الطلبات الخارجية الوحيدة هي تلك المُرسَلة إلى Google YouTube Data API v3

## 🛠️ التفاصيل التقنية

| التفاصيل | المعلومات |
|---|---|
| المكدس التقني | HTML وCSS وJavaScript نقي — بدون أطر عمل |
| الخادم الخلفي | لا يوجد — يعمل بالكامل على جانب العميل |
| API | YouTube Data API v3 |
| النتائج | حتى 50 نتيجة، مرتبة حسب التاريخ |
| فلتر التاريخ | يستخدم معاملات API ‏`publishedAfter` / `publishedBefore` |
| فلتر المدة | يستخدم معامل API ‏`videoDuration` ‏(`short` / `medium` / `long`) |
| فلتر Shorts | على جانب العميل — يزيل أي نتيجة تحتوي على `#` في العنوان أو الوصف |
| السمات | خصائص CSS المخصصة (`var(--accent)`) يتم تبديلها عبر JavaScript |
| الجسيمات | قيم `animation-delay` سالبة لتُملأ الجسيمات مسبقًا عند التحميل |
| التخزين | `localStorage` لمفتاح API وتفضيل السمة |
