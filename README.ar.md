# 📺 أداة البحث في يوتيوب

أداة ويب للبحث عن مقاطع الفيديو والبث المباشر على يوتيوب بالترتيب الزمني الحقيقي باستخدام مفتاح API الخاص بك. لا يلزم تسجيل الدخول — فقط الصق مفتاحك وابحث.

## **أداة البحث في يوتيوب:** <https://slotchy.github.io/YouTube-Search-Tool>
## **درس تعليمي لمفتاح API:** <https://www.youtube.com/watch?v=eE5WGiwqlFE>

---

## ✨ الميزات

* 🔍 ابحث عن مقاطع الفيديو أو البث المباشر مرتبةً بالأحدث أولاً
* 📅 تصفية حسب التاريخ — اختر يوماً واحداً أو نطاقاً مخصصاً باستخدام تقويم تفاعلي
* 🕐 فلتر الساعة — تضييق النتائج إلى نافذة زمنية محددة (يحول تلقائياً منطقتك الزمنية إلى UTC)
* 🚫 تصفية Shorts — إخفاء مقاطع الفيديو التي تحتوي على علامات تصنيف (`#`) في عنوانها أو وصفها
* ⏱️ فلتر مدة الفيديو — في وضع الفيديو، صفّ حسب قصير (<4 دقائق) أو متوسط (4–20 دقيقة) أو طويل (20+ دقيقة)
* 🎨 8 ثيمات لونية — بنفسجي، سماوي، زمردي، وردي، كهرماني، أزرق، زهري وليموني
* 💾 يتم حفظ مفتاح API وثيم اللون في متصفحك تلقائياً
* ✨ خلفية جسيمات متحركة
* 🕒 يعرض وقت النشر لكل نتيجة
* 🖱️ انقر على أي نتيجة لفتح الفيديو مباشرةً
* ⭐ أضف أي فيديو إلى المفضلة بنقرة واحدة — تستمر المفضلة عبر الجلسات
* 📋 أضف ملاحظات لأي فيديو — انقر على زر Notes في البطاقة لتوسيع منطقة النص، يتم الحفظ تلقائياً
* 📄 صدّر جلسة البحث الكاملة كملف `.md` — يشمل جميع النتائج من كل عملية بحث مع الروابط وصور مصغرة والمفضلة والملاحظات، بدون حصة إضافية

> **ملاحظة:** وضع البث المباشر معطّل عند تفعيل تصفية التاريخ. فلتر مدة الفيديو متاح فقط في وضع الفيديو.

## 🔑 كيفية الحصول على مفتاح YouTube API

1. افتح تبويباً جديداً، انتقل إلى [Gmail](https://mail.google.com) وسجّل الدخول إلى حساب Google
2. انتقل إلى [console.cloud.google.com](https://console.cloud.google.com)
3. اقبل **شروط الخدمة** وانقر على **أوافق وأتابع**
4. في أعلى يسار الصفحة، انقر على **تحديد مشروع**، ثم انقر على **مشروع جديد**
5. أعطِ مشروعك اسماً وانقر على **إنشاء** — انتظر حتى يكتمل
6. افتح **قائمة التنقل** (ثلاثة أشرطة في أعلى اليسار)، انتقل إلى **واجهات برمجة التطبيقات → المكتبة**
7. ابحث عن **YouTube Data API v3** وانقر عليه
8. انقر على **تفعيل** وانتظر حتى يكتمل التفعيل
9. على اليسار، انقر على **بيانات الاعتماد**
10. في الأعلى، انقر على **إنشاء بيانات اعتماد → مفتاح API → إنشاء**
11. بعد إنشاء المفتاح، انقر على **مفتاح API** لفتح إعداداته
12. تحت **قيود API**، انقر على **تقييد المفتاح → تحديد APIs**
    - إذا لم تظهر YouTube Data API v3 فوراً، أعد تحميل الصفحة
    - اختر **YouTube Data API v3** وانقر على **موافق**
13. انقر على **حفظ**
14. انقر على **إظهار المفتاح**، ثم انسخه
15. انتقل إلى [أداة البحث في يوتيوب](https://slotchy.github.io/YouTube-Search-Tool)، الصق مفتاحك وأجرِ بحثاً للتأكد من أنه يعمل

> **الحصة:** يوفر المستوى المجاني 10,000 وحدة/يوم. كل بحث يكلف ~100 وحدة (~100 بحث/يوم مجاناً).

## 🚀 الاستخدام

1. الصق مفتاح YouTube API الخاص بك — يتم حفظه تلقائياً للزيارات المستقبلية
2. اختر **مقاطع الفيديو** أو **البث المباشر**
3. اكتب موضوع البحث
4. *(اختياري)* اختر **مدة الفيديو** للتصفية حسب المدة (وضع الفيديو فقط)
5. *(اختياري)* حدد **تصفية Shorts** لاستبعاد مقاطع الفيديو ذات العلامات
6. *(اختياري)* فعّل **تصفية حسب التاريخ** واختر نطاقاً في التقويم
7. *(اختياري)* إذا تم اختيار تاريخ، حدد ساعة **البداية** و**النهاية**
8. انقر على **البحث عن الأحدث** (أو **بحث** عند تفعيل تصفية التاريخ)
9. انقر على أي بطاقة نتيجة لفتح الفيديو
10. *(اختياري)* انقر على **☆ Favorite** في أي بطاقة لحفظها كمفضلة
11. *(اختياري)* انقر على **📋 Notes** لتوسيع منطقة الملاحظات وكتابة ملاحظة
12. انقر على **Export .MD** لتنزيل جلستك الكاملة كملف Markdown
13. استخدم ألواح الألوان على اليمين لتغيير الثيم — يتم تذكره بين الزيارات

## 📄 التصدير

يظهر زر التصدير تلقائياً بعد أول بحث ويجمع النتائج طوال الجلسة بأكملها.

يتم تنظيم ملف `.md` على النحو التالي:

* **⭐ المفضلة** — جميع مقاطع الفيديو المُنجَّمة، مدرجة أولاً
* **📝 الملاحظات** — جميع مقاطع الفيديو التي تحتوي على ملاحظات
* **جميع النتائج** — كل نتيجة مجمعة حسب مصطلح البحث

تستمر المفضلة والملاحظات في متصفحك عبر تحديثات الصفحة عبر `localStorage`.

---

## 🔒 الخصوصية

* يتم تخزين مفتاح API محلياً في متصفحك (`localStorage`) — لا يُرسل إلى أي مكان إلا Google
* يتم تخزين ثيم اللون والمفضلة والملاحظات محلياً
* لا يوجد خادم خلفي — تتم جميع المعالجة على جانب العميل
* الطلبات الخارجية الوحيدة هي إلى Google YouTube Data API v3

## 🛠️ التفاصيل التقنية

| التفاصيل | المعلومات |
| --- | --- |
| المكدس التقني | HTML وCSS وJavaScript خالص — لا أطر عمل |
| الخادم الخلفي | لا يوجد — كامل على جانب العميل |
| API | YouTube Data API v3 |
| النتائج | حتى 50 لكل بحث، مرتبة حسب التاريخ |
| تصفية التاريخ | يستخدم معاملات `publishedAfter` / `publishedBefore` |
| تصفية المدة | يستخدم معامل `videoDuration` |
| تصفية Shorts | من جانب العميل — يزيل النتائج التي تحتوي على `#` |
| التخصيص | خصائص CSS مخصصة يتم تبديلها عبر JavaScript |
| الجسيمات | قيم `animation-delay` سالبة لملء الشاشة عند التحميل |
| التخزين | `localStorage` لمفتاح API والثيم والمفضلة والملاحظات |
| التصدير | تنزيل Blob من جانب العميل — لا استدعاءات API إضافية |
