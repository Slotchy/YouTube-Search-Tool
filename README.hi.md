# 📺 YouTube सर्च टूल

एक वेब-आधारित टूल जो आपकी अपनी YouTube API कुंजी का उपयोग करके YouTube वीडियो और लाइवस्ट्रीम को वास्तविक कालानुक्रमिक क्रम में खोजता है। कोई साइन-इन आवश्यक नहीं — बस अपनी कुंजी पेस्ट करें और खोजें।

## **YouTube सर्च टूल:** https://slotchy.github.io/YouTube-Search-Tool

---

## ✨ विशेषताएं

* 🔍 वीडियो या लाइवस्ट्रीम को सबसे हाल के पहले क्रम से खोजें
* 📅 तारीख से फ़िल्टर करें — इंटरैक्टिव कैलेंडर से एक दिन या कस्टम तारीख सीमा चुनें
* 🕐 घंटे का फ़िल्टर — चुनी गई तारीख के भीतर एक विशिष्ट समय विंडो में परिणाम सीमित करें (आपके स्थानीय टाइमज़ोन को स्वचालित रूप से UTC में बदलता है)
* 🚫 Shorts फ़िल्टर करें — उन वीडियो को छुपाएं जिनके शीर्षक या विवरण में हैशटैग (`#`) हों
* ⏱️ वीडियो लंबाई फ़िल्टर — वीडियो मोड में, छोटे (<4 मिनट), मध्यम (4–20 मिनट) या लंबे (20+ मिनट) से फ़िल्टर करें
* 🎨 8 रंग थीम — स्वैच पैनल के माध्यम से बैंगनी, सियान, एमरल्ड, रोज़, एम्बर, नीला, पिंक और लाइम
* 💾 API कुंजी और रंग थीम आपके ब्राउज़र में स्वचालित रूप से सहेजी जाती है
* ✨ एनिमेटेड पार्टिकल बैकग्राउंड, लोड होते ही भरा हुआ
* 🕒 प्रत्येक परिणाम के लिए समय-पूर्व दिखाता है
* 🖱️ सीधे वीडियो खोलने के लिए किसी भी परिणाम पर क्लिक करें

> **नोट:** तारीख फ़िल्टरिंग सक्रिय होने पर लाइवस्ट्रीम मोड अक्षम हो जाता है। वीडियो लंबाई फ़िल्टर केवल वीडियो मोड में उपलब्ध है।

---

## 🔑 YouTube API कुंजी कैसे प्राप्त करें

1. [console.cloud.google.com](https://console.cloud.google.com) पर जाएं
2. एक नया प्रोजेक्ट बनाएं या कोई मौजूदा चुनें
3. **YouTube Data API v3** सक्षम करें
4. **Credentials → Create Credentials → API Key** पर जाएं
5. अपनी API कुंजी कॉपी करें और टूल में पेस्ट करें

> **कोटा:** मुफ़्त स्तर प्रतिदिन 10,000 यूनिट प्रदान करता है। प्रत्येक खोज लगभग 100 यूनिट खर्च करती है (मुफ़्त में ~100 खोजें/दिन)।

---

## 🚀 उपयोग कैसे करें

1. अपनी YouTube API कुंजी पेस्ट करें — भविष्य की विज़िट के लिए स्वचालित रूप से सहेजी जाती है
2. **वीडियो** या **लाइवस्ट्रीम** चुनें
3. अपना खोज विषय टाइप करें
4. *(वैकल्पिक)* अवधि से फ़िल्टर करने के लिए **वीडियो लंबाई** चुनें (केवल वीडियो मोड)
5. *(वैकल्पिक)* शीर्षक या विवरण में हैशटैग वाले वीडियो बाहर करने के लिए **Shorts फ़िल्टर करें** जांचें
6. *(वैकल्पिक)* **तारीख से फ़िल्टर करें** चालू करें और कैलेंडर पर दिन या तारीख सीमा चुनें
7. *(वैकल्पिक)* तारीख चुनने पर, दिन के समय के अनुसार फ़िल्टर करने के लिए **शुरू** और **अंत** घंटा सेट करें
8. **सबसे नए खोजें** क्लिक करें (या तारीख फ़िल्टरिंग चालू होने पर **खोजें**)
9. वीडियो खोलने के लिए किसी भी परिणाम कार्ड पर क्लिक करें
10. थीम बदलने के लिए दाईं ओर के रंग स्वैच उपयोग करें — विज़िट के बीच याद रखा जाता है

## 🔒 गोपनीयता

* API कुंजी आपके ब्राउज़र में स्थानीय रूप से (`localStorage`) संग्रहीत — Google को भेजने के अलावा कहीं नहीं भेजी जाती
* रंग थीम प्राथमिकता आपके ब्राउज़र में स्थानीय रूप से संग्रहीत
* कोई बैकएंड नहीं — सभी प्रसंस्करण क्लाइंट-साइड होता है
* केवल बाहरी अनुरोध Google YouTube Data API v3 को किए जाते हैं

## 🛠️ तकनीकी विवरण

| विवरण | जानकारी |
|---|---|
| स्टैक | शुद्ध HTML, CSS, JavaScript — कोई फ्रेमवर्क नहीं |
| बैकएंड | कोई नहीं — पूरी तरह क्लाइंट-साइड |
| API | YouTube Data API v3 |
| परिणाम | 50 तक, तारीख के अनुसार क्रमबद्ध |
| तारीख फ़िल्टरिंग | API के `publishedAfter` / `publishedBefore` पैरामीटर उपयोग करता है |
| अवधि फ़िल्टरिंग | API के `videoDuration` पैरामीटर उपयोग करता है (`short` / `medium` / `long`) |
| Shorts फ़िल्टरिंग | क्लाइंट-साइड — शीर्षक या विवरण में `#` वाले परिणाम हटाता है |
| थीमिंग | JavaScript द्वारा CSS कस्टम प्रॉपर्टी (`var(--accent)`) बदली जाती है |
| पार्टिकल्स | नकारात्मक `animation-delay` मान ताकि पार्टिकल्स लोड पर प्री-पॉपुलेट हों |
| स्टोरेज | API कुंजी और थीम प्राथमिकता के लिए `localStorage` |
