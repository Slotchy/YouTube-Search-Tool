# 📺 YouTube Search Tool

एक वेब-आधारित टूल जो आपकी अपनी YouTube API कुंजी का उपयोग करके YouTube वीडियो और लाइवस्ट्रीम को सही कालानुक्रमिक क्रम में खोजता है। साइन इन की आवश्यकता नहीं — बस अपनी कुंजी पेस्ट करें और खोजें।

## ✨ विशेषताएं

* 🔍 वीडियो या लाइवस्ट्रीम को सबसे हाल के पहले क्रमबद्ध करके खोजें
* 📅 तारीख के अनुसार फ़िल्टर करें — इंटरैक्टिव कैलेंडर का उपयोग करके एक दिन या कस्टम तारीख सीमा चुनें
* 🕐 घंटे का फ़िल्टर — चुनी हुई तारीख के भीतर एक विशिष्ट समय विंडो तक परिणामों को सीमित करें (आपके स्थानीय समय क्षेत्र को स्वचालित रूप से UTC में बदलता है)
* 🚫 Shorts फ़िल्टर करें — उन वीडियो को छुपाता है जिनके शीर्षक या विवरण में हैशटैग (`#`) हैं
* ⏱️ वीडियो अवधि फ़िल्टर — Videos चुने जाने पर, Short (<4 मिनट), Medium (4–20 मिनट), या Long (20+ मिनट) के अनुसार फ़िल्टर करें
* 🎨 8 रंग थीम — स्वॉच पैनल के माध्यम से Purple, Cyan, Emerald, Rose, Amber, Blue, Pink और Lime
* 💾 API कुंजी और रंग थीम स्वचालित रूप से आपके ब्राउज़र में सहेजी जाती है
* ✨ एनिमेटेड पार्टिकल बैकग्राउंड, लोड होने पर पहले से भरा हुआ
* 🕒 प्रत्येक परिणाम के लिए कितने समय पहले प्रकाशित हुआ दिखाता है
* 🖱️ किसी भी परिणाम पर क्लिक करके सीधे वीडियो खोलें

> **नोट:** तारीख फ़िल्टरिंग सक्रिय होने पर Livestreams मोड अक्षम हो जाता है। वीडियो अवधि फ़िल्टर केवल Videos मोड में उपलब्ध है।

## 🔑 YouTube API कुंजी कैसे प्राप्त करें

1. एक नया टैब खोलें, [Gmail](https://mail.google.com) पर जाएं और अपने Google अकाउंट में साइन इन करें
2. [console.cloud.google.com](https://console.cloud.google.com) पर जाएं
3. जब पूछा जाए, **सेवा की शर्तें** स्वीकार करें और **सहमत हैं और जारी रखें** पर क्लिक करें
4. पृष्ठ के ऊपर बाईं ओर **प्रोजेक्ट चुनें** पर क्लिक करें, फिर उस विंडो के ऊपर दाईं ओर **नया प्रोजेक्ट** पर क्लिक करें
5. अपने प्रोजेक्ट को नाम दें और **बनाएं** पर क्लिक करें — इसके बनने की प्रतीक्षा करें
6. **नेविगेशन मेनू** (ऊपर बाईं ओर तीन बार) खोलें, **APIs और सेवाएं** तक नीचे स्क्रॉल करें और **लाइब्रेरी** पर क्लिक करें
7. नीचे स्क्रॉल करें जब तक **YouTube Data API v3** न मिले और उस पर क्लिक करें
8. **सक्षम करें** पर क्लिक करें और पूरी तरह से सक्रिय होने तक प्रतीक्षा करें
9. बाईं ओर **क्रेडेंशियल** पर क्लिक करें
10. ऊपर **क्रेडेंशियल बनाएं → API कुंजी** पर क्लिक करें, फिर **बनाएं** पर क्लिक करें
11. कुंजी बनने के बाद, उसकी सेटिंग खोलने के लिए **API कुंजी** पर क्लिक करें
12. **API प्रतिबंध** के अंतर्गत, **कुंजी प्रतिबंधित करें** पर क्लिक करें, फिर **APIs चुनें** पर क्लिक करें
    - यदि YouTube Data API v3 तुरंत नहीं दिखती, तो पृष्ठ को रीफ्रेश करें
    - नीचे स्क्रॉल करें, **YouTube Data API v3** चुनें और **ठीक है** पर क्लिक करें
13. **सहेजें** पर क्लिक करें
14. **कुंजी दिखाएं** पर क्लिक करें, फिर अपनी कुंजी कॉपी करें
15. [YouTube Search Tool](https://slotchy.github.io/YouTube-Search-Tool) पर जाएं, अपनी कुंजी पेस्ट करें और यह सुनिश्चित करने के लिए खोज चलाएं कि यह काम कर रही है

> **कोटा:** मुफ़्त स्तर प्रतिदिन 10,000 इकाइयां प्रदान करता है। प्रत्येक खोज में ~100 इकाइयां लगती हैं (~100 खोजें/दिन मुफ़्त)।

## 🚀 उपयोग

1. अपनी YouTube API कुंजी पेस्ट करें — भविष्य की विज़िट के लिए स्वचालित रूप से सहेजी जाती है
2. **Videos** या **Livestreams** चुनें
3. अपना खोज विषय टाइप करें
4. *(वैकल्पिक)* अवधि के अनुसार फ़िल्टर करने के लिए **वीडियो अवधि** चुनें (केवल Videos मोड)
5. *(वैकल्पिक)* शीर्षक या विवरण में हैशटैग वाले वीडियो बाहर करने के लिए **Shorts फ़िल्टर करें** चेक करें
6. *(वैकल्पिक)* **तारीख के अनुसार फ़िल्टर करें** टॉगल करें और कैलेंडर पर एक दिन या तारीख सीमा चुनें
7. *(वैकल्पिक)* यदि कोई तारीख चुनी गई है, तो दिन के समय के अनुसार फ़िल्टर करने के लिए **शुरू** और **अंत** घंटे सेट करें
8. **सबसे हाल के खोजें** पर क्लिक करें (या तारीख फ़िल्टरिंग चालू होने पर **खोजें**)
9. वीडियो खोलने के लिए किसी भी परिणाम कार्ड पर क्लिक करें
10. एक्सेंट थीम बदलने के लिए दाईं ओर रंग स्वॉच का उपयोग करें — विज़िट के बीच याद रखा जाता है

## 🔒 गोपनीयता

* API कुंजी आपके ब्राउज़र में स्थानीय रूप से संग्रहीत (`localStorage`) — Google के अलावा कहीं नहीं भेजी जाती
* रंग थीम प्राथमिकता आपके ब्राउज़र में स्थानीय रूप से संग्रहीत
* कोई बैकएंड नहीं — सभी प्रोसेसिंग क्लाइंट-साइड होती है
* केवल बाहरी अनुरोध Google YouTube Data API v3 को भेजे जाते हैं

| विवरण | जानकारी |
| --- | --- |
| स्टैक | शुद्ध HTML, CSS, JavaScript — कोई फ्रेमवर्क नहीं |
| बैकएंड | कोई नहीं — पूरी तरह क्लाइंट-साइड |

| पार्टिकल | नकारात्मक `animation-delay` मान ताकि पार्टिकल लोड पर पहले से भर जाएं |
| स्टोरेज | API कुंजी और थीम प्राथमिकता के लिए `localStorage` |
