# 📺 YouTube सर्च टूल

एक वेब-आधारित टूल जो आपकी अपनी YouTube API कुंजी का उपयोग करके YouTube वीडियो और लाइवस्ट्रीम को सही कालानुक्रमिक क्रम में खोजता है। साइन-इन की आवश्यकता नहीं — बस अपनी कुंजी पेस्ट करें और खोजें।

## **YouTube सर्च टूल:** <https://slotchy.github.io/YouTube-Search-Tool>
## **API कुंजी ट्यूटोरियल:** <https://www.youtube.com/watch?v=eE5WGiwqlFE>

## ✨ विशेषताएं

* 🔍 सबसे हाल के वीडियो या लाइवस्ट्रीम पहले खोजें
* 📅 तारीख से फ़िल्टर करें — इंटरेक्टिव कैलेंडर का उपयोग करके एक दिन या कस्टम दिनांक सीमा चुनें
* 🕐 घंटे का फ़िल्टर — चुनी हुई तारीख के भीतर एक विशिष्ट समय विंडो तक परिणाम सीमित करें (स्थानीय टाइमज़ोन को UTC में स्वतः परिवर्तित करता है)
* 🚫 Shorts फ़िल्टर करें — उन वीडियो को छुपाएं जिनके शीर्षक या विवरण में हैशटैग (`#`) हैं
* ⏱️ वीडियो लंबाई फ़िल्टर — वीडियो मोड में, छोटे (<4 मिनट), मध्यम (4–20 मिनट) या लंबे (20+ मिनट) के अनुसार फ़िल्टर करें
* 🎨 8 रंग थीम — बैंगनी, सियान, पन्ना, गुलाब, एम्बर, नीला, गुलाबी और नींबू
* 💾 API कुंजी और रंग थीम आपके ब्राउज़र में स्वतः सहेजे जाते हैं
* ✨ एनिमेटेड पार्टिकल बैकग्राउंड
* 🕒 प्रत्येक परिणाम के लिए समय पहले दिखाता है
* 🖱️ वीडियो सीधे खोलने के लिए किसी भी परिणाम पर क्लिक करें
* ⭐ एक टैप से किसी भी वीडियो को पसंदीदा बनाएं — पसंदीदा सत्रों में बने रहते हैं
* 📋 किसी भी वीडियो में नोट्स जोड़ें — टेक्स्ट एरिया खोलने के लिए कार्ड पर Notes बटन टैप करें, स्वचालित रूप से सहेजा जाता है
* 📄 अपना पूरा सर्च सत्र `.md` फ़ाइल के रूप में निर्यात करें — URL, थंबनेल लिंक, पसंदीदा और नोट्स के साथ, बिना किसी अतिरिक्त कोटा के

> **नोट:** तारीख फ़िल्टर सक्रिय होने पर लाइवस्ट्रीम मोड अक्षम है। वीडियो लंबाई फ़िल्टर केवल वीडियो मोड में उपलब्ध है।

## 🔑 YouTube API कुंजी कैसे प्राप्त करें

1. एक नया टैब खोलें, [Gmail](https://mail.google.com) पर जाएं और अपने Google खाते में साइन इन करें
2. [console.cloud.google.com](https://console.cloud.google.com) पर जाएं
3. **सेवा की शर्तें** स्वीकार करें और **सहमत हैं और जारी रखें** पर क्लिक करें
4. ऊपर बाईं ओर **प्रोजेक्ट चुनें** पर क्लिक करें, फिर **नया प्रोजेक्ट** पर क्लिक करें
5. अपने प्रोजेक्ट को नाम दें और **बनाएं** पर क्लिक करें — पूरा होने की प्रतीक्षा करें
6. **नेविगेशन मेनू** खोलें (ऊपर बाईं ओर तीन बार), **APIs और Services → लाइब्रेरी** पर जाएं
7. **YouTube Data API v3** खोजें और उस पर क्लिक करें
8. **सक्षम करें** पर क्लिक करें और पूरी तरह सक्रिय होने की प्रतीक्षा करें
9. बाईं ओर **क्रेडेंशियल** पर क्लिक करें
10. ऊपर **क्रेडेंशियल बनाएं → API कुंजी → बनाएं** पर क्लिक करें
11. कुंजी जनरेट होने के बाद, **API कुंजी** पर क्लिक करके इसकी सेटिंग खोलें
12. **API प्रतिबंध** के तहत, **कुंजी प्रतिबंधित करें → APIs चुनें** पर क्लिक करें
    - यदि YouTube Data API v3 तुरंत दिखाई नहीं देता, तो पेज रीफ्रेश करें
    - **YouTube Data API v3** चुनें और **ठीक है** पर क्लिक करें
13. **सहेजें** पर क्लिक करें
14. **कुंजी दिखाएं** पर क्लिक करें, फिर अपनी कुंजी कॉपी करें
15. [YouTube सर्च टूल](https://slotchy.github.io/YouTube-Search-Tool) पर जाएं, अपनी कुंजी पेस्ट करें और काम करने की पुष्टि के लिए एक खोज चलाएं

> **कोटा:** निःशुल्क स्तर 10,000 यूनिट/दिन प्रदान करता है। प्रत्येक खोज ~100 यूनिट खर्च करती है।

## 🚀 उपयोग

1. अपनी YouTube API कुंजी पेस्ट करें — भविष्य की यात्राओं के लिए स्वतः सहेजा जाता है
2. **वीडियो** या **लाइवस्ट्रीम** चुनें
3. अपना खोज विषय टाइप करें
4. *(वैकल्पिक)* अवधि के अनुसार फ़िल्टर करने के लिए **वीडियो लंबाई** चुनें
5. *(वैकल्पिक)* Shorts को बाहर करने के लिए **Shorts फ़िल्टर करें** चेक करें
6. *(वैकल्पिक)* **तारीख से फ़िल्टर** टॉगल करें और कैलेंडर पर दिनांक चुनें
7. *(वैकल्पिक)* यदि तारीख चुनी है, तो **शुरू** और **अंत** घंटा सेट करें
8. **सबसे नए खोजें** पर क्लिक करें (तारीख फ़िल्टर चालू होने पर **खोजें**)
9. वीडियो खोलने के लिए किसी भी रिजल्ट कार्ड पर क्लिक करें
10. *(वैकल्पिक)* किसी भी कार्ड पर **☆ Favorite** क्लिक करके पसंदीदा सहेजें
11. *(वैकल्पिक)* नोट्स एरिया खोलने के लिए **📋 Notes** क्लिक करें
12. अपना पूरा सत्र Markdown फ़ाइल के रूप में डाउनलोड करने के लिए **Export .MD** क्लिक करें
13. थीम बदलने के लिए दाईं ओर रंग स्वैच का उपयोग करें — यात्राओं के बीच याद रखा जाता है

## 📄 निर्यात

पहली खोज के बाद निर्यात बटन स्वचालित रूप से दिखाई देता है और पूरे सत्र में परिणाम जमा करता है।

`.md` फ़ाइल इस प्रकार व्यवस्थित है:

* **⭐ पसंदीदा** — सभी स्टार किए गए वीडियो, पहले सूचीबद्ध
* **📝 नोट्स** — नोट्स वाले सभी वीडियो
* **सभी परिणाम** — खोज शब्द के अनुसार समूहीकृत प्रत्येक परिणाम

पसंदीदा और नोट्स `localStorage` के माध्यम से पेज रीफ्रेश के बीच बने रहते हैं।

## 🔒 गोपनीयता

* API कुंजी आपके ब्राउज़र में स्थानीय रूप से संग्रहीत (`localStorage`) — Google के अलावा कहीं नहीं भेजी जाती
* रंग थीम, पसंदीदा और नोट्स स्थानीय रूप से संग्रहीत
* कोई बैकएंड नहीं — सभी प्रोसेसिंग क्लाइंट-साइड होती है
* केवल बाहरी अनुरोध Google YouTube Data API v3 को किए जाते हैं
