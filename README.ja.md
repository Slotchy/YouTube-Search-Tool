# 📺 YouTube Search Tool

自分のYouTube APIキーを使用して、YouTubeの動画とライブストリームを真の時系列順で検索するWebツールです。サインイン不要 — キーを貼り付けて検索するだけです。

## **YouTube Search Tool:** <https://slotchy.github.io/YouTube-Search-Tool>
## **APIキー チュートリアル:** <https://www.youtube.com/watch?v=eE5WGiwqlFE>

---

## ✨ 機能

* 🔍 最新順に並べられた動画またはライブストリームを検索
* 📅 日付でフィルター — インタラクティブなカレンダーを使って1日またはカスタム日付範囲を選択
* 🕐 時間フィルター — 選択した日付内の特定の時間帯に結果を絞り込む（ローカルタイムゾーンを自動的にUTCに変換）
* 🚫 Shortsをフィルタリング — タイトルや説明にハッシュタグ（`#`）を含む動画を非表示
* ⏱️ 動画の長さフィルター — Videosモードで、短い（<4分）、普通（4〜20分）、長い（20分以上）でフィルタリング
* 🎨 8つのカラーテーマ — スウォッチパネルでパープル、シアン、エメラルド、ローズ、アンバー、ブルー、ピンク、ライムを選択
* 💾 APIキーとカラーテーマがブラウザに自動保存
* ✨ アニメーションパーティクル背景、読み込み時に事前表示
* 🕒 各結果の投稿からの経過時間を表示
* 🖱️ 結果をクリックして動画を直接開く

> **注意:** 日付フィルタリングが有効な場合、ライブストリームモードは無効になります。動画の長さフィルターはVideosモードでのみ使用できます。

## 🔑 YouTube APIキーの取得方法

1. 新しいタブを開き、[Gmail](https://mail.google.com)にアクセスしてGoogleアカウントにサインインする
2. [console.cloud.google.com](https://console.cloud.google.com)に移動する
3. 求められたら、**利用規約**に同意して**同意して続行**をクリックする
4. ページの左上で**プロジェクトを選択**をクリックし、そのウィンドウの右上で**新しいプロジェクト**をクリックする
5. プロジェクトに名前を付けて**作成**をクリック — 作成が完了するまで待つ
6. **ナビゲーションメニュー**（左上の三本線）を開き、**APIとサービス**までスクロールして**ライブラリ**をクリックする
7. **YouTube Data API v3**が見つかるまでスクロールしてクリックする
8. **有効にする**をクリックして完全に有効化されるまで待つ
9. 左側で**認証情報**をクリックする
10. 上部で**認証情報を作成 → APIキー**をクリックし、**作成**をクリックする
11. キーが生成されたら、設定を開くために**APIキー**をクリックする
12. **APIの制限**で**キーを制限**をクリックし、**APIを選択**をクリックする
    - YouTube Data API v3がすぐに表示されない場合は、ページを更新する
    - スクロールして**YouTube Data API v3**を選択し、**OK**をクリックする
13. **保存**をクリックする
14. **キーを表示**をクリックし、キーをコピーする
15. [YouTube Search Tool](https://slotchy.github.io/YouTube-Search-Tool)にアクセスし、キーを貼り付けて検索を実行して動作を確認する

> **クォータ:** 無料プランでは1日10,000ユニットが提供されます。各検索は約100ユニットを消費します（無料で1日約100回検索可能）。

## 🚀 使い方

1. YouTube APIキーを貼り付ける — 次回以降の訪問のために自動保存される
2. **Videos**または**Livestreams**を選択する
3. 検索トピックを入力する
4. *（任意）* 時間でフィルタリングするには**動画の長さ**を選択（Videosモードのみ）
5. *（任意）* タイトルや説明にハッシュタグを含む動画を除外するには**Shortsをフィルター**にチェックを入れる
6. *（任意）* **日付でフィルター**を切り替えてカレンダーで1日または日付範囲を選択する
7. *（任意）* 日付が選択されている場合、時間帯でフィルタリングするには**開始**と**終了**の時間を設定する
8. **最新を検索**をクリック（日付フィルタリングがオンの場合は**検索**）
9. 結果カードをクリックして動画を開く
10. 右側のカラースウォッチを使ってアクセントテーマを変更 — 訪問間で記憶される

---

## 🔒 プライバシー

* APIキーはブラウザにローカルで保存（`localStorage`）— Google以外のどこにも送信されない
* カラーテーマの設定はブラウザにローカルで保存
* バックエンドなし — すべての処理はクライアント側で実行
* 外部リクエストはGoogle YouTube Data API v3へのみ

## 🛠️ 技術詳細

| 詳細 | 情報 |
| --- | --- |
| スタック | 純粋なHTML、CSS、JavaScript — フレームワークなし |
| バックエンド | なし — 完全にクライアントサイド |
| API | YouTube Data API v3 |
| 結果数 | 最大50件、日付順 |
| 日付フィルタリング | `publishedAfter` / `publishedBefore` APIパラメーターを使用 |
| 時間フィルタリング | `videoDuration` APIパラメーターを使用（`short` / `medium` / `long`） |
| Shortsフィルタリング | クライアントサイド — タイトルや説明に`#`を含む結果を削除 |
| テーマ | JavaScriptでCSSカスタムプロパティ（`var(--accent)`）を切り替え |
| パーティクル | 負の`animation-delay`値で読み込み時にパーティクルを事前表示 |
| ストレージ | APIキーとテーマ設定に`localStorage`を使用 |
