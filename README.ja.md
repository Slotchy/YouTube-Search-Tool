# 📺 YouTube 検索ツール

自分の YouTube API キーを使って YouTube 動画やライブ配信を本当の時系列順で検索できる Web ツールです。ログイン不要 — キーを貼り付けるだけで検索できます。

## **YouTube 検索ツール：** <https://slotchy.github.io/YouTube-Search-Tool>
## **API キーチュートリアル：** <https://www.youtube.com/watch?v=eE5WGiwqlFE>

---

## ✨ 機能

* 🔍 最新順に並んだ動画またはライブ配信を検索
* 📅 日付でフィルター — インタラクティブなカレンダーで単日またはカスタム日付範囲を選択
* 🕐 時間フィルター — 選択した日付内の特定の時間帯に結果を絞り込む（ローカルタイムゾーンを UTC に自動変換）
* 🚫 ショートをフィルター — タイトルや説明にハッシュタグ（`#`）を含む動画を非表示
* ⏱️ 動画の長さフィルター — 動画モードで短い（<4分）・中程度（4〜20分）・長い（20分以上）でフィルター
* 🎨 8 つのカラーテーマ — パープル、シアン、エメラルド、ローズ、アンバー、ブルー、ピンク、ライム
* 💾 API キーとカラーテーマがブラウザに自動保存
* ✨ アニメーションパーティクル背景
* 🕒 各結果の投稿からの経過時間を表示
* 🖱️ 任意の結果をクリックして動画を直接開く
* ⭐ ワンタップで任意の動画をお気に入りに追加 — セッション間で持続保存
* 📋 任意の動画にメモを追加 — カード上の Notes ボタンをタップしてテキストエリアを展開、自動保存
* 📄 検索セッション全体を `.md` ファイルとしてエクスポート — すべての検索結果、URL、サムネイルリンク、お気に入り、メモを含み、追加クォータなし

> **注意：** 日付フィルターが有効な場合、ライブ配信モードは無効になります。動画の長さフィルターは動画モードでのみ利用可能です。

## 🔑 YouTube API キーの取得方法

1. 新しいタブを開き、[Gmail](https://mail.google.com) にアクセスして Google アカウントにサインイン
2. [console.cloud.google.com](https://console.cloud.google.com) に移動
3. **利用規約**に同意して**同意して続行**をクリック
4. 左上の**プロジェクトを選択**をクリックし、次に**新しいプロジェクト**をクリック
5. プロジェクトに名前を付けて**作成**をクリック — 完了するまで待つ
6. **ナビゲーションメニュー**（左上の三本線）を開き、**API とサービス → ライブラリ**に移動
7. **YouTube Data API v3** を見つけてクリック
8. **有効にする**をクリックし、完全に有効化されるまで待つ
9. 左側の**認証情報**をクリック
10. 上部の**認証情報を作成 → API キー → 作成**をクリック
11. キーが生成されたら、**API キー**をクリックして設定を開く
12. **API の制限**で**キーを制限 → API を選択**をクリック
    - YouTube Data API v3 がすぐに表示されない場合はページを更新
    - **YouTube Data API v3** を選択して**OK** をクリック
13. **保存**をクリック
14. **キーを表示**をクリックしてコピー
15. [YouTube 検索ツール](https://slotchy.github.io/YouTube-Search-Tool)にアクセスし、キーを貼り付けて検索を実行して動作を確認

> **割り当て：** 無料枠は 10,000 ユニット/日。各検索は約 100 ユニット消費（無料で約 100 回/日）。

## 🚀 使い方

1. YouTube API キーを貼り付ける — 将来の訪問のために自動保存
2. **動画**または**ライブ配信**を選択
3. 検索トピックを入力
4. *（任意）* **動画の長さ**を選択して時間でフィルター（動画モードのみ）
5. *（任意）* **ショートをフィルター**にチェックしてハッシュタグ付き動画を除外
6. *（任意）* **日付でフィルター**を切り替えてカレンダーで日付を選択
7. *（任意）* 日付を選択した場合、**開始**と**終了**時間を設定
8. **最新を検索**をクリック（日付フィルター有効時は**検索**）
9. 任意の結果カードをクリックして動画を開く
10. *（任意）* 任意のカードの **☆ Favorite** をクリックしてお気に入りに保存
11. *（任意）* **📋 Notes** をクリックしてメモエリアを展開してメモを入力
12. **Export .MD** をクリックしてセッション全体を Markdown ファイルとしてダウンロード
13. 右側のカラースウォッチを使ってテーマを変更 — 訪問間で記憶される

## 📄 エクスポート

エクスポートボタンは最初の検索後に自動的に表示され、セッション全体で結果を蓄積します。

`.md` ファイルの構成：

* **⭐ お気に入り** — スター付きのすべての動画、最初にリスト
* **📝 メモ** — メモが添付されたすべての動画
* **すべての結果** — 検索語ごとにグループ化された各結果

お気に入りとメモは `localStorage` を通じてページのリフレッシュ間でブラウザに保持されます。

---

## 🔒 プライバシー

* API キーはブラウザにローカル保存（`localStorage`）— Google 以外には送信されない
* カラーテーマ、お気に入り、メモはローカルに保存
* バックエンドなし — すべての処理はクライアントサイドで実行
* 外部リクエストは Google YouTube Data API v3 のみ

## 🛠️ 技術詳細

| 詳細 | 情報 |
| --- | --- |
| スタック | 純粋な HTML、CSS、JavaScript — フレームワークなし |
| バックエンド | なし — 完全クライアントサイド |
| API | YouTube Data API v3 |
| 結果 | 検索ごとに最大 50 件、日付順 |
| 日付フィルター | `publishedAfter` / `publishedBefore` API パラメータを使用 |
| 時間フィルター | `videoDuration` API パラメータを使用 |
| ショートフィルター | クライアントサイド — タイトルや説明に `#` を含む結果を除去 |
| テーマ | JavaScript で切り替える CSS カスタムプロパティ |
| パーティクル | 負の `animation-delay` 値でロード時に画面を事前に満たす |
| ストレージ | API キー、テーマ、お気に入り、メモに `localStorage` |
| エクスポート | クライアントサイド Blob ダウンロード — 追加 API コールなし |
