# 📺 YouTube 検索ツール

自分の YouTube API キーを使って YouTube の動画とライブ配信を本当の時系列順で検索できるウェブベースのツールです。ログイン不要 — キーを貼り付けて検索するだけです。

## **YouTube 検索ツール：** https://slotchy.github.io/YouTube-Search-Tool

---

## ✨ 機能

* 🔍 動画またはライブ配信を最新順で検索
* 📅 日付でフィルター — インタラクティブなカレンダーで1日または任意の日付範囲を選択
* 🕐 時間フィルター — 選択した日付内の特定の時間帯に結果を絞り込む（ローカルタイムゾーンを自動的にUTCに変換）
* 🚫 Shorts をフィルター — タイトルまたは説明にハッシュタグ（`#`）を含む動画を非表示
* ⏱️ 動画の長さフィルター — 動画モード選択時に、ショート（4分未満）、中程度（4〜20分）、長い（20分以上）でフィルタリング
* 🎨 8つのカラーテーマ — スウォッチパネルからパープル、シアン、エメラルド、ローズ、アンバー、ブルー、ピンク、ライムを選択
* 💾 API キーとカラーテーマがブラウザに自動保存
* ✨ アニメーションパーティクル背景、読み込み時にプリフィル
* 🕒 各結果の経過時間を表示
* 🖱️ 任意の結果をクリックして動画を直接開く

> **注意：** 日付フィルタリングが有効な場合、ライブ配信モードは無効になります。動画の長さフィルターは動画モードでのみ使用できます。

---

## 🔑 YouTube API キーの取得方法

1. [console.cloud.google.com](https://console.cloud.google.com) にアクセス
2. 新しいプロジェクトを作成するか、既存のものを選択
3. **YouTube Data API v3** を有効化
4. **認証情報 → 認証情報を作成 → API キー** に移動
5. API キーをコピーしてツールに貼り付け

> **割り当て：** 無料ティアでは1日10,000ユニットが提供されます。各検索は約100ユニット消費します（無料で1日約100回検索可能）。

---

## 🚀 使い方

1. YouTube API キーを貼り付け — 次回以降の訪問に向けて自動保存されます
2. **動画**または**ライブ配信**を選択
3. 検索トピックを入力
4. *（任意）* 動画の長さでフィルタリングするには**動画の長さ**を選択（動画モードのみ）
5. *（任意）* タイトルまたは説明にハッシュタグを含む動画を除外するには**Shorts をフィルター**にチェック
6. *（任意）* **日付でフィルター**を切り替え、カレンダーで日またはデータ範囲を選択
7. *（任意）* 日付を選択した場合、時間帯でフィルタリングするために**開始**と**終了**の時間を設定
8. **最新を検索**をクリック（日付フィルタリング有効時は**検索**）
9. 任意の結果カードをクリックして動画を開く
10. 右側のカラースウォッチを使用してテーマを変更 — 訪問間で記憶されます

---

## 🔒 プライバシー

* API キーはブラウザのローカルに保存（`localStorage`）— Google 以外には送信されません
* カラーテーマの設定はブラウザのローカルに保存
* バックエンドなし — すべての処理はクライアントサイドで行われます
* 外部リクエストは Google YouTube Data API v3 へのみ

---

## 🛠️ 技術的な詳細

| 詳細 | 情報 |
|---|---|
| スタック | 純粋な HTML、CSS、JavaScript — フレームワークなし |
| バックエンド | なし — 完全なクライアントサイド |
| API | YouTube Data API v3 |
| 結果数 | 最大50件、日付順 |
| 日付フィルタリング | API の `publishedAfter` / `publishedBefore` パラメータを使用 |
| 時間フィルタリング | API の `videoDuration` パラメータを使用（`short` / `medium` / `long`） |
| Shorts フィルタリング | クライアントサイド — タイトルまたは説明に `#` を含む結果を削除 |
| テーマ設定 | JavaScript で CSS カスタムプロパティ（`var(--accent)`）を切り替え |
| パーティクル | 負の `animation-delay` 値で読み込み時にパーティクルをプリポピュレート |
| ストレージ | API キーとテーマ設定に `localStorage` を使用 |
