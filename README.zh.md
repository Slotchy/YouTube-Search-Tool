# 📺 YouTube 搜索工具

一款基于网页的工具，使用您自己的 YouTube API 密钥以真正的时间顺序搜索 YouTube 视频和直播。无需登录 — 只需粘贴您的密钥即可搜索。

## **YouTube 搜索工具：** https://slotchy.github.io/YouTube-Search-Tool

---

## ✨ 功能特点

* 🔍 按最新优先排序搜索视频或直播
* 📅 按日期筛选 — 使用交互式日历选择单天或自定义日期范围
* 🕐 小时筛选 — 将结果缩小到所选日期内的特定时间窗口（自动将您的本地时区转换为 UTC）
* 🚫 过滤 Shorts — 隐藏标题或描述中包含话题标签（`#`）的视频
* ⏱️ 视频时长筛选 — 选择视频模式时，可按短片（<4 分钟）、中等（4–20 分钟）或长视频（20+ 分钟）筛选
* 🎨 8 种颜色主题 — 通过色块面板选择紫色、青色、翠绿、玫瑰、琥珀、蓝色、粉红和青柠
* 💾 API 密钥和颜色主题自动保存在您的浏览器中
* ✨ 带动画效果的粒子背景，加载时预填充
* 🕒 显示每个结果的发布时间
* 🖱️ 点击任意结果直接打开视频

> **注意：** 日期筛选激活时，直播模式将被禁用。视频时长筛选仅在视频模式下可用。

---

## 🔑 如何获取 YouTube API 密钥

1. 前往 [console.cloud.google.com](https://console.cloud.google.com)
2. 创建新项目或选择现有项目
3. 启用 **YouTube Data API v3**
4. 导航至 **凭据 → 创建凭据 → API 密钥**
5. 复制您的 API 密钥并粘贴到工具中

> **配额：** 免费层每天提供 10,000 个单位。每次搜索消耗约 100 个单位（免费每天约可搜索 100 次）。

---

## 🚀 使用方法

1. 粘贴您的 YouTube API 密钥 — 自动保存供以后使用
2. 选择**视频**或**直播**
3. 输入您的搜索主题
4. *（可选）* 选择**视频时长**按时长筛选（仅限视频模式）
5. *（可选）* 勾选**过滤 Shorts** 以排除标题或描述中含话题标签的视频
6. *（可选）* 开启**按日期筛选**并在日历上选择一天或日期范围
7. *（可选）* 选择日期后，设置**开始**和**结束**小时以按时段筛选
8. 点击**查找最新**（日期筛选开启时显示**搜索**）
9. 点击任意结果卡片打开视频
10. 使用右侧的色块更改主题 — 访问之间自动记住

---

## 🔒 隐私说明

* API 密钥存储在您的浏览器本地（`localStorage`）— 除了发送给 Google 外，不会发送到任何地方
* 颜色主题偏好存储在您的浏览器本地
* 无后端 — 所有处理均在客户端完成
* 唯一的外部请求是向 Google YouTube Data API v3 发起的

---

## 🛠️ 技术细节

| 详情 | 说明 |
|---|---|
| 技术栈 | 纯 HTML、CSS、JavaScript — 无框架 |
| 后端 | 无 — 完全客户端运行 |
| API | YouTube Data API v3 |
| 结果数量 | 最多 50 条，按日期排序 |
| 日期筛选 | 使用 API 的 `publishedAfter` / `publishedBefore` 参数 |
| 时长筛选 | 使用 API 的 `videoDuration` 参数（`short` / `medium` / `long`） |
| Shorts 过滤 | 客户端处理 — 删除标题或描述中含 `#` 的结果 |
| 主题切换 | 通过 JavaScript 替换 CSS 自定义属性（`var(--accent)`） |
| 粒子效果 | 使用负数 `animation-delay` 值使粒子在页面加载时预填充 |
| 存储 | 使用 `localStorage` 保存 API 密钥和主题偏好 |
