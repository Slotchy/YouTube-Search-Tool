# 📺 YouTube 搜索工具

一款基于网页的工具，使用您自己的 YouTube API 密钥按真实时间顺序搜索 YouTube 视频和直播。无需登录 — 只需粘贴您的密钥即可搜索。

## **YouTube 搜索工具：** <https://slotchy.github.io/YouTube-Search-Tool>
## **API 密钥教程：** <https://www.youtube.com/watch?v=eE5WGiwqlFE>

---

## ✨ 功能

* 🔍 按最新优先搜索视频或直播
* 📅 按日期过滤 — 使用交互式日历选择单天或自定义日期范围
* 🕐 小时过滤 — 将结果缩小到所选日期内的特定时间窗口（自动将本地时区转换为 UTC）
* 🚫 过滤 Shorts — 隐藏标题或描述中包含话题标签（`#`）的视频
* ⏱️ 视频时长过滤 — 在视频模式下按短片（<4 分钟）、中等（4–20 分钟）或长视频（20+ 分钟）过滤
* 🎨 8 种颜色主题 — 紫色、青色、翠绿、玫瑰、琥珀、蓝色、粉色和青柠
* 💾 API 密钥和颜色主题自动保存在浏览器中
* ✨ 动态粒子背景，加载时预填充
* 🕒 显示每个结果的发布时间
* 🖱️ 点击任意结果直接打开视频
* ⭐ 一键收藏任何视频 — 收藏夹跨会话持久保存
* 📋 为任何视频添加笔记 — 点击卡片上的笔记按钮展开文本区域，自动保存
* 📄 将整个搜索会话导出为 `.md` 文件 — 包含每次搜索的所有结果、URL、缩略图链接、收藏和笔记，无需额外配额

> **注意：** 日期过滤激活时，直播模式不可用。视频时长过滤仅在视频模式下可用。

## 🔑 如何获取 YouTube API 密钥

1. 打开新标签页，前往 [Gmail](https://mail.google.com) 并登录您的 Google 账号
2. 导航至 [console.cloud.google.com](https://console.cloud.google.com)
3. 接受**服务条款**并点击**同意并继续**
4. 在页面左上角，点击**选择项目**，然后点击**新建项目**
5. 为项目命名并点击**创建** — 等待创建完成
6. 打开**导航菜单**（左上角三条横线），前往 **API 和服务 → 库**
7. 向下滚动找到 **YouTube Data API v3** 并点击
8. 点击**启用**并等待完全激活
9. 在左侧点击**凭据**
10. 在顶部点击**创建凭据 → API 密钥 → 创建**
11. 密钥生成后，点击 **API 密钥**打开其设置
12. 在 **API 限制**下，点击**限制密钥 → 选择 API**
    - 如果 YouTube Data API v3 没有立即出现，请刷新页面
    - 选择 **YouTube Data API v3** 并点击**确定**
13. 点击**保存**
14. 点击**显示密钥**，然后复制您的密钥
15. 前往 [YouTube 搜索工具](https://slotchy.github.io/YouTube-Search-Tool)，粘贴您的密钥并运行搜索以确认其正常工作

> **配额：** 免费层每天提供 10,000 个单位。每次搜索消耗约 100 个单位（免费每天约 100 次搜索）。

## 🚀 使用方法

1. 粘贴您的 YouTube API 密钥 — 自动保存以供将来访问
2. 选择**视频**或**直播**
3. 输入搜索主题
4. *（可选）* 选择**视频时长**按时长过滤（仅视频模式）
5. *（可选）* 勾选**过滤 Shorts** 以排除带话题标签的视频
6. *（可选）* 切换**按日期过滤**并在日历上选择日期范围
7. *（可选）* 选择日期后，设置**开始**和**结束**小时
8. 点击**查找最新**（日期过滤开启时为**搜索**）
9. 点击任意结果卡片打开视频
10. *（可选）* 点击卡片上的 **☆ Favorite** 收藏视频
11. *（可选）* 点击 **📋 Notes** 展开笔记区域并输入笔记
12. 点击 **Export .MD**（浮动按钮）将整个会话下载为 Markdown 文件
13. 使用右侧颜色色板更改主题 — 跨访问记忆

## 📄 导出

搜索后导出按钮自动出现，并在整个会话中累积结果。

`.md` 文件结构如下：

* **⭐ 收藏** — 所有已收藏的视频，优先列出
* **📝 笔记** — 所有附有笔记的视频
* **所有结果** — 按搜索词分组的每个结果

收藏和笔记通过 `localStorage` 在页面刷新之间持久保存在浏览器中。

---

## 🔒 隐私

* API 密钥存储在浏览器本地（`localStorage`）— 除 Google 外不发送至任何地方
* 颜色主题、收藏和笔记存储在浏览器本地
* 无后端 — 所有处理均在客户端进行
* 唯一的外部请求是发送至 Google YouTube Data API v3

## 🛠️ 技术详情

| 详情 | 信息 |
| --- | --- |
| 技术栈 | 纯 HTML、CSS、JavaScript — 无框架 |
| 后端 | 无 — 完全客户端 |
| API | YouTube Data API v3 |
| 结果 | 每次搜索最多 50 个，按日期排序 |
| 日期过滤 | 使用 `publishedAfter` / `publishedBefore` API 参数 |
| 时长过滤 | 使用 `videoDuration` API 参数（`short` / `medium` / `long`）|
| Shorts 过滤 | 客户端 — 移除标题或描述中含 `#` 的结果 |
| 主题 | CSS 自定义属性（`var(--accent)`）通过 JavaScript 切换 |
| 粒子 | 负 `animation-delay` 值使粒子在加载时预填充屏幕 |
| 存储 | `localStorage` 用于 API 密钥、主题、收藏和笔记 |
| 导出 | 客户端 Blob 下载 — 无额外 API 调用，零配额消耗 |
