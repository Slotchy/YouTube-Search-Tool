# 📺 YouTube Search Tool

一款基于网页的工具，使用您自己的 YouTube API 密钥按真实时间顺序搜索 YouTube 视频和直播。无需登录 — 只需粘贴您的密钥并搜索。

## **YouTube Search Tool：** <https://slotchy.github.io/YouTube-Search-Tool>
## **API 密钥教程：** <https://www.youtube.com/watch?v=eE5WGiwqlFE>

---

## ✨ 功能特点

* 🔍 搜索视频或直播，按最新排序
* 📅 按日期筛选 — 使用交互式日历选择单天或自定义日期范围
* 🕐 小时筛选 — 将结果缩小到所选日期内的特定时间窗口（自动将您的本地时区转换为 UTC）
* 🚫 过滤短视频 — 隐藏标题或描述中包含话题标签（`#`）的视频
* ⏱️ 视频时长筛选 — 在视频模式下，按短（<4 分钟）、中（4–20 分钟）或长（20+ 分钟）筛选
* 🎨 8 种颜色主题 — 通过色板面板选择紫色、青色、翠绿、玫瑰、琥珀、蓝色、粉色和青柠
* 💾 API 密钥和颜色主题自动保存在您的浏览器中
* ✨ 动态粒子背景，加载时预填充
* 🕒 显示每个结果的发布时间
* 🖱️ 点击任意结果直接打开视频

> **注意：** 日期筛选启用时，直播模式不可用。视频时长筛选仅在视频模式下可用。

## 🔑 如何获取 YouTube API 密钥

1. 打开新标签页，前往 [Gmail](https://mail.google.com) 并登录您的 Google 账号
2. 导航至 [console.cloud.google.com](https://console.cloud.google.com)
3. 出现提示时，接受**服务条款**并点击**同意并继续**
4. 在页面左上角点击**选择项目**，然后在该窗口右上角点击**新建项目**
5. 为您的项目命名并点击**创建** — 等待创建完成
6. 打开**导航菜单**（左上角的三条横线），向下滚动到 **API 和服务**，点击**库**
7. 向下滚动找到 **YouTube Data API v3** 并点击
8. 点击**启用**并等待其完全激活
9. 在左侧点击**凭据**
10. 在顶部点击**创建凭据 → API 密钥**，然后点击**创建**
11. 密钥生成后，点击 **API 密钥**打开其设置
12. 在 **API 限制**下，点击**限制密钥**，然后点击**选择 API**
    - 如果 YouTube Data API v3 没有立即显示，请刷新页面
    - 向下滚动，选择 **YouTube Data API v3**，点击**确定**
13. 点击**保存**
14. 点击**显示密钥**，然后复制您的密钥
15. 前往 [YouTube Search Tool](https://slotchy.github.io/YouTube-Search-Tool)，粘贴您的密钥并进行搜索以确认其正常工作

> **配额：** 免费套餐每天提供 10,000 个单位。每次搜索消耗约 100 个单位（每天约 100 次免费搜索）。

## 🚀 使用方法

1. 粘贴您的 YouTube API 密钥 — 自动保存以供将来访问
2. 选择**视频**或**直播**
3. 输入您的搜索主题
4. *（可选）* 选择**视频时长**按时长筛选（仅视频模式）
5. *（可选）* 勾选**过滤短视频**以排除标题或描述中含话题标签的视频
6. *（可选）* 切换**按日期筛选**并在日历上选择一天或日期范围
7. *（可选）* 如果选择了日期，设置**开始**和**结束**小时以按时段筛选
8. 点击**查找最新**（日期筛选开启时点击**搜索**）
9. 点击任意结果卡片打开视频
10. 使用右侧的色板更改主题颜色 — 访问间自动记忆

## 🔒 隐私

* API 密钥存储在您的浏览器本地（`localStorage`）— 除 Google 外不会发送到任何地方
* 颜色主题偏好存储在您的浏览器本地
* 无后端 — 所有处理均在客户端进行
* 唯一的外部请求是发送至 Google YouTube Data API v3

## 🛠️ 技术详情

| 详情 | 说明 |
| --- | --- |
| 技术栈 | 纯 HTML、CSS、JavaScript — 无框架 |
| 后端 | 无 — 完全客户端 |
| API | YouTube Data API v3 |
| 结果数 | 最多 50 条，按日期排序 |
| 日期筛选 | 使用 `publishedAfter` / `publishedBefore` API 参数 |
| 时长筛选 | 使用 `videoDuration` API 参数（`short` / `medium` / `long`） |
| 短视频筛选 | 客户端 — 移除标题或描述中含 `#` 的结果 |
| 主题 | 通过 JavaScript 切换 CSS 自定义属性（`var(--accent)`） |
| 粒子 | 使用负 `animation-delay` 值使粒子在加载时预填充 |
| 存储 | `localStorage` 用于 API 密钥和主题偏好 |
