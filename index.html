🔑 How to Get a YouTube API Key

  1. 1 Open a new tab, go to Gmail and sign into your Google account
  2. 2 Go to console.cloud.google.com
  3. 3 Accept the Terms of Service and click Agree and Continue
  4. 4 Click Select a Project (top left), then New Project (top right of that window)
  5. 5 Name your project and click Create — wait for it to finish
  6. 6 Open the ☰ Navigation Menu (top left), scroll to APIs & Services → Library
  7. 7 Scroll down and click YouTube Data API v3, then click Enable
  8. 8 Click Credentials on the left, then Create Credentials → API Key → Create
  9. 9 Click the generated API Key to open its settings
  10. 10 Under API Restrictions, click Restrict Key → Select APIs (refresh if it doesn't appear)
  11. 11 Select YouTube Data API v3, click OK, then Save
  12. 12 Click Show Key, copy it, and paste it above
