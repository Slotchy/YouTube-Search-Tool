🔑 How to Get a YouTube API Key

  1. 1 Go to console.cloud.google.com
  2. 2 Create a new project or select an existing one
  3. 3 Click the ☰ Navigation Menu (top left)
  4. 4 Go to APIs & Services → Library
  5. 5 Search for and click YouTube Data API v3
  6. 6 Click Enable
  7. 7 Open the ☰ Navigation Menu again and go to APIs & Services → Credentials
  8. 8 Click Create Credentials → API Key
  9. 9 Copy your key and paste it above
